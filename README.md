# 🏗️ Structura AI Insights

> AI-powered infrastructure health monitoring platform that monitors bridge safety across India using satellite, seismic, weather, and traffic data — without physical sensors.

---

## 📌 About the Project

**Structura AI** is an intelligent infrastructure monitoring system that leverages:
- 🛰️ **Satellite & Seismic Data** — Remote sensing for structural analysis
- 🌦️ **Weather & Traffic Data** — Environmental and load monitoring
- 🤖 **AI Crack Detection** — Computer vision for structural damage assessment
- 🧬 **Digital Twins** — Virtual replicas of physical bridges for simulation
- 🌐 **Multilingual Insights** — Dashboard available in multiple Indian languages

---

## 🚀 Getting Started

### Prerequisites

Make sure you have **Node.js** and **npm** installed.
- [Install Node.js with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Step 1: Clone the repository
git clone https://github.com/AnishaRihana19A/structura-ai-insights.git

# Step 2: Navigate into the project folder
cd structura-ai-insights/structura-ai-insights-main

# Step 3: Install dependencies
npm install

# Step 4: Start the development server
npm run dev

```
